## **📚 Documentation**

### **API Reference**
```python
from quantum_shield.core import QuantumRadarSimulator
from quantum_shield.hardware import NVDiamondInterface
from quantum_shield.processing import QuantumFusionEngine

# Initialize quantum radar
radar = QuantumRadarSimulator(
    wavelength=1.55e-6,      # 1550 nm
    aperture=0.1,            # 10 cm aperture
    quantum_advantage=4.0    # 6 dB advantage
)

# Run simulation
result = radar.detect_target(
    cross_section=0.1,       # m²
    range=1000,              # meters
    integration_time=0.1     # seconds
)

print(f"Detection probability: {result['detection_probability']:.3f}")
print(f"Quantum advantage: {result['quantum_advantage_db']:.2f} dB")
```

### **Hardware Interface Example**
```python
from quantum_shield.hardware.real_interfaces import QuantumSensorArray

# Initialize sensor array
sensors = QuantumSensorArray(
    num_sensors=4,
    sensor_type='nv_diamond',
    spacing=1e-3  # 1 mm spacing
)

# Connect to hardware
if sensors.connect(ip_address='192.168.1.100'):
    # Take measurement
    data = sensors.measure_quantum_state(
        integration_time=1.0,
        microwave_freq=2.87e9
    )
    
    # Process quantum data
    processed = sensors.process_quantum_data(
        data,
        error_correction='surface_code',
        decoherence_compensation=True
    )
```
